# Sheet Music TV (UniApp, Android TV)

## 当前功能

- 列表页直接平铺显示 `static/scores` 下所有图片（不显示目录）
- 缩略图统一尺寸，支持遥控器焦点移动与进入预览
- 预览页左右切图，图片按屏幕居中完整显示
- 预览页节拍器红绿灯显示（四拍同一音高）
- 预览页确认键可稳定暂停/继续（防重复触发）
- 列表页提供二维码，手机扫码进入管理台
- 手机管理台支持对列表排序、上传图片并实时同步到大屏

## 目录约定

将乐谱图片直接放在：

- `static/scores/*.jpg`
- 或 `static/scores/<任意子目录>/*.jpg`

不需要再执行 `generate:sheets` 才能显示列表。

## 启动方式（H5 联调）

1. 启动 uni-app H5 开发服务（你当前方式）
2. 另开一个终端启动控制服务：

```bash
npm run control:server
```

控制服务默认端口：`9091`。

## 手机扫码管理

1. 打开大屏列表页，看到二维码和管理链接
2. 手机扫码（或直接访问管理链接）
3. 在手机管理台中：
- 上移/下移调整图片顺序（实时同步）
- 选择图片上传（实时追加到大屏列表）

说明：上传图片在当前会话内生效（无数据库持久化）。

## 遥控器/键盘

### 列表页

- `上/下/左/右`：焦点移动
- `确认`：进入预览

### 预览页

- `左/右`：上一张/下一张
- `上/下`：BPM +1 / -1
- `确认`：节拍器开始/暂停
- `菜单`：打开节拍器设置面板
- `返回`：退出预览
